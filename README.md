# ReasonML : modules

Installation (no global installs necessary):

```text
cd 5ServiceEchanges/
npm install
```

Build : 

```text
npm run build
```

Automatic compilation:

```text
cd 5ServiceEchanges/
npm run watch
```text

Run one of the files:

```text
cd 5ServiceEchangess/
node src/HelloWorld.bs.js
```
